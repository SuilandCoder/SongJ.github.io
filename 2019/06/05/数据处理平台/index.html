数据处理平台

发表于 2019-06-05 更新于 2022-08-20 分类于 项目
之后一段时间将会着手做这个平台，主要功能包括：数据转换服务、数据可视化服务以及地理数据分析服务。

1.为什么开发这个平台？

两方面的原因促使我开发这个平台,

  • 一是看了超然哥的硕士毕业论文后，觉得自己很有必要先写一篇像样的论文，以保证硕士毕业的时候不会难产，所以我想在Online_Saga的基础上做一个功能更完善的平台，然后写一篇论文；
  • 二是为通用的（易扩展的）模型比较平台打基础，要想实现多个模型输入输出的比较，少不了数据转换服务和数据可视化服务，所以先用这个平台实现一波这两个服务。

2.平台功能

2.1 地理数据分析服务

主要包括一些saga的地理分析工具：

  • climate
  • grid
  • imagery
  • shapes
  • simulation
  • spatial and geostatistics
  • terrain analysis
